Как получить Webhook для СберЧат
Обычно для СберЧат есть два способа отправки сообщений:

Через Webhook-бота (проще):

Зайди в СберЧат → Создать бота → Получить Webhook URL

Вставь его в SBERCHAT_WEBHOOK_URL

Через API бота (если бот уже есть):

Нужен токен бота и ID чата

Вставь их в SBERCHAT_BOT_TOKEN и SBERCHAT_CHAT_ID

Если не знаешь, как получить — обратись к администратору СберЧат в вашей компании.
